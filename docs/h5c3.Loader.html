Class: Loader

h5c3. Loader

new Loader()

[Extends h5c3.Base]

The Loader takes care of loading resources (downloading) and then notifying you when everything is ready. The loader is a static class that will always be constructed by the engine and accessible through th h5c3.device.loader member.

Using the loader you can load h5c3.Image's, h5c3.DataResources's, and h5c3.Sound's.

Typically you use the loader from within your game class onReady method (called automatically by the engine). 


TheGame = h5c3.Game.extend('TheGame',
{},
{
    onReady:function ()
    {
        this._super(); // call the base class' onReady

        // disable caching when developing
        if (h5c3.devMode===true)
            h5c3.device.loader.setDisableCache();

        // load up resources
        h5c3.device.loader.add(new h5c3.Image('spaceship', 'images/spaceship.png'));

        // fire up the loader (with a callback once done)
        h5c3.device.loader.start(this.onLoading.bind(this), this.onLoaded.bind(this));
    },

    onLoading:function (percentageComplete)
    {
        // display progress, such as a loading bar
    },

    onLoaded:function ()
    {
        // we're ready; make the magic happen
    }
});
You can disable caching using setDisableCache. This is the default when in devMode (when the engine has not been packed/minified.

h5c3. Loader

new Loader()

[Extends h5c3.Base]

The Loader takes care of loading resources (downloading) and then notifying you when everything is ready. The loader is a static class that will always be constructed by the engine and accessible through th h5c3.device.loader member.

Using the loader you can load h5c3.Image's, h5c3.DataResources's, and h5c3.Sound's.

Typically you use the loader from within your game class onReady method (called automatically by the engine). 


TheGame = h5c3.Game.extend('TheGame',
{},
{
    onReady:function ()
    {
        this._super(); // call the base class' onReady

        // disable caching when developing
        if (h5c3.devMode===true)
            h5c3.device.loader.setDisableCache();

        // load up resources
        h5c3.device.loader.add(new h5c3.Image('spaceship', 'images/spaceship.png'));

        // fire up the loader (with a callback once done)
        h5c3.device.loader.start(this.onLoading.bind(this), this.onLoaded.bind(this));
    },

    onLoading:function (percentageComplete)
    {
        // display progress, such as a loading bar
    },

    onLoaded:function ()
    {
        // we're ready; make the magic happen
    }
});
You can disable caching using setDisableCache. This is the default when in devMode (when the engine has not been packed/minified.
