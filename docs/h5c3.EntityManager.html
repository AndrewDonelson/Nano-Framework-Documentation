Class: EntityManager

h5c3. EntityManager

new EntityManager()

[Extends h5c3.Base]

Manages entities in a layer. This is the primary entity manager for the entity system. It contains, indexes and handles the lifecycle of all entities. Unless you are building your own systems in a complex way, you should be using the h5c3.EntityLayer to handle general entity management.

h5c3. EntityManager

new EntityManager()

[Extends h5c3.Base]

Manages entities in a layer. This is the primary entity manager for the entity system. It contains, indexes and handles the lifecycle of all entities. Unless you are building your own systems in a complex way, you should be using the h5c3.EntityLayer to handle general entity management.
