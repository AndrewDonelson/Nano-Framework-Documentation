Class: Spatial

Spatial

new Spatial()

[Extends h5c3.components.Component]
[Used in h5c3.systems.Render, h5c3.systems.Physics, h5c3.systems.Layout]

Represents where an entity exists in 2D space (x, y, width and height). This component is mostly for use by other systems to update and use.
