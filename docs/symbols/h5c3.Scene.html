Class Detail
h5c3.Scene()
[Extends h5c3.Base]

A game is primarily a container for various "scenes", with each scene containing one or more layers. You can construct a scene, and use addScene to add it to the game. This is typically done once all the queued resources have been loaded: 


onLoaded:function ()
{
   // construct the game scene
   this.gameScene = new GameScene();

   // add it to the game
   this.addScene(this.gameScene);
}
Active scenes will be updated and drawn by the system, inactive ones will not. Adding a scene makes it active by default.

To activate a scene (such as displaying a menu scene): 


myGame.activateScene(myMenuScene);
You can likewise deactivate a scene (it will no longer be rendered or processed): 

myGame.deactivateScene(myMenuScene);
Upon activating a scene, the game's onSceneActivated is called passing in the scene that became active. Likewise onSceneDeactivated will be called when a scene is deactivated.

You can access scenes by calling getFirstScene or getFirstActiveScene which will return a h5c3.LinkedListNode you can use to loop through the list of scenes: 


var sceneNode = myGame.getFirstScene();
while (sceneNode)
{
   var scene = sceneNode.object();
   // scene.doSomething();

   // move to the next one (will be null if done)
   sceneNode = sceneNode.next();
}

setViewPort(x, y, width, height)
Sets the view port to the given top left postion (x, y) and dimensions (width and height) The view port represents the on-screen pixels dimensions of the game relative to the associated canvas. Use the view port dimensions to render different scenes at different positions on screen. e.g. a game layer would typically be 0, 0, canvas.width, canvas.height whereas a mini map may just be in the top left corner of the screen (0, 0, 100, 100).
Parameters:
{Number} x
X position to render the scene within the canvas (in screen pixels)
{Number} y
Y position to render the scene within the canvas (in screen pixels)
{Number} width
The maximum width to render (in screen pixels)
{Number} height
The maximum height to render (in screen pixels)
