h5c3.HTMLVariables

This class allow you to access and change any part of an HTML document.  It does not use the data properties of HTML5; to keep maximum compatability it is accomplish by adding a class calledc variable to any element and this will give you access to that said element.

<div id="someDiv class="variable"></div>

then, in your application you can

//Array of all variables in document
var allVars = h5c3.vars.list();

// Get the value of variable title int he document
var title =h5c3.vars.get(title);
