Class: Color

h5c3. Color

new Color()

A general purpose representation of a color. You can create a color either an array [R,G,B] or using a hex color string in the form of #RRGGBB. (For alpha see the h5c3.component.Alpha and h5c3.components.Fade).

Create a color using either new: 


var color = new h5c3.Color([255, 0, 0]); // super red
var color2 = new h5c3.Color('#00FF00'); // super green

Or, as a pooled object: 


var color = h5c3.Color.create([255, 0, 0]); // super red
